Parameter API Labs

Workshop-aligned exercises for manifest parameters, runtime parameters, and custom settings pages.

Repository-Based Labs

These labs follow the official parameter workshop path: declare known parameters, manage dynamic parameters, then expose values through a custom UI and VAPIX.

GitHub: axis-acap-tip-workshop/parameter
Lab 1: Manifest Parameter
Declare a parameter in manifest.json, register a callback, and update the value through VAPIX.
Beginner
35 min

Learning Objectives

  • Understand how paramConfig creates an application parameter
  • Open the app parameter scope with ax_parameter_new
  • Register a callback for ParameterManifest
  • Run a GLib main loop so callbacks are delivered
Final Capability

After these labs, students should be able to choose between manifest parameters, runtime parameters, and a custom UI backed by VAPIX param.cgi and AXParameter callbacks.
