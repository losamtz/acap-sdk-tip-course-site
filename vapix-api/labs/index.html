Lab 1: Dynamic Overlay with VAPIX
Learn to use VAPIX for creating dynamic overlays using D-Bus credentials, cURL, and Jansson JSON processing
Beginner • 75 min

Learning Objectives

  • Understand VAPIX service account authentication via D-Bus
  • Implement HTTP requests using cURL to local virtual host
  • Parse JSON responses using Jansson library
  • Create and manage dynamic overlays through VAPIX

Key Concepts

  • g_dbus_proxy_call_sync() for credential retrieval
  • curl_easy_setopt() for HTTP configuration
  • json_loads() and json_dumps() for JSON processing
  • Dynamic overlay API endpoint integration
Lab Completion Guide
Tips for successful completion and common troubleshooting

Success Tips:

  • Complete Lab 1 before attempting Lab 2 for foundational knowledge
  • Test D-Bus permissions thoroughly before proceeding
  • Use curl verbose mode for debugging HTTP/SOAP requests
  • Validate XML/SOAP structure before sending requests

Common Issues:

  • D-Bus permission denied errors
  • cURL timeout on local virtual host
  • SOAP envelope malformed or missing namespaces
  • JSON/XML parsing failures with malformed responses
