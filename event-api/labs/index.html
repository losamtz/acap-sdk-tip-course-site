Repository-Based Labs

These hands-on labs use the official Axis ACAP TIP Workshop repository. Each lab builds upon the previous one, progressing from basic event production to advanced subscriber implementations with real-world event handling scenarios.

GitHub: axis-acap-tip-workshop/event
Lab 1: Event Producer Implementation
Master event production with four progressive exercises covering different event types and data structures
Beginner • 120 min

Learning Objectives

  • Understand event producer architecture and responsibilities
  • Initialize and configure the ACAP Event API handler
  • Declare custom event types with proper key-value structures
  • Send events with structured data to the event system

Key Concepts

  • ax_event_handler_new() for event handler initialization
  • Event declaration with topic hierarchy
  • Key-value data structures with ax_event_key_value_set_new()
  • GLib main loop integration for event handling
Lab Completion Guide
Tips for successful completion and common troubleshooting

Success Tips:

  • Complete Lab 1 exercises in sequence before moving to Lab 2
  • Test producer and subscriber applications together
  • Use the provided debugging tools and logging
  • Review the theory section if concepts are unclear

Common Issues:

  • Event handler not properly initialized
  • Callback functions blocking the event loop
  • Incorrect event declaration syntax
  • Memory leaks from not freeing event objects
