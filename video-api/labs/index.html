Lab 1: Discover Channels And Resolutions
Use VDO channel APIs to discover available streams, supported resolutions, and stream metadata.
Beginner
35 min

Learning Objectives

  • List VDO channels and identify the channel id used by later examples
  • Query supported resolutions with a VdoMap filter
  • Create a stream object only to inspect stream info
  • Read rotation before choosing requested width and height
Final Capability

After these labs, students should be ready to understand why Larod can consume VDO-backed fds and why overlay2 applications must manage VDO stream events explicitly.
