Lab 1: Basic Overlay Development
Learn fundamental overlay concepts by implementing basic shapes and text overlays using axoverlay and Cairo
Beginner • 90 min

Learning Objectives

  • Understand axoverlay library initialization and setup
  • Create basic overlays with normalized coordinates
  • Use Cairo graphics for drawing rectangles and text
  • Handle different color spaces (4-bit palette vs ARGB32)

Key Concepts

  • axoverlay_new() for overlay creation
  • Normalized coordinate system (0.0 to 1.0)
  • Cairo surface management with cairo_create()
  • Render callback integration
