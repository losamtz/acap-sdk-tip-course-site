Overlay API Labs

Workshop-aligned exercises for axoverlay and Cairo. Dynamic overlay via VAPIX is intentionally excluded.

Repository-based labs
These labs follow the four axoverlay samples in the workshop: draw-rectangle, draw-text, add-logo, and draw-views.
axis-acap-tip-workshop/overlay
Lab 1: draw-rectangle
Draw a centered rectangle with the Cairo backend and 4-bit palette colorspace.
Beginner - 45 min
overlay/draw-rectangle
Tasks
  • - Initialize axoverlay with AXOVERLAY_CAIRO_IMAGE_BACKEND.
  • - Configure palette colors with axoverlay_set_palette_color.
  • - Use max camera resolution for overlay width and height.
  • - Draw a centered rectangle in render_overlay_cb.
  • - Use adjustment_cb to adapt dimensions when stream rotation changes.
Validation
  • A yellow rectangle appears centered on the stream.
  • The overlay dimensions follow stream width/height and rotation.
  • The application destroys the overlay and calls axoverlay_cleanup on exit.
