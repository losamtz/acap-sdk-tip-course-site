Lab 1: CivetWeb Reverse Proxy Implementation
Build a complete reverse proxy web application using CivetWeb with AXParameter integration and JSON API endpoints
Intermediate • 90 min

Learning Objectives

  • Initialize and configure CivetWeb server with proper options
  • Implement GET and POST handlers for API endpoints
  • Integrate AXParameter management with error handling
  • Create RESTful JSON endpoints using Jansson library

Key Concepts

  • mg_start() for server initialization
  • HTTP request handler registration and routing
  • JSON processing with json_object()
  • Reverse proxy configuration and manifest setup
