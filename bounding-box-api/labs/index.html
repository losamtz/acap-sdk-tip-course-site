Lab 1: Single View Bounding Box
Learn the fundamentals of BBox API by implementing a static bounding box on a single camera view
Beginner • 45 min

Learning Objectives

  • Understand BBox API session lifecycle
  • Master coordinate system selection
  • Implement proper styling and colors
  • Handle resource management and cleanup

Key Concepts

  • bbox_view_new() for single view sessions
  • Scene vs frame normalized coordinates
  • Deferred rendering with bbox_commit()
  • GLib main loop integration
